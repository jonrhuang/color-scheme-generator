# Color Scheme Generator
Generates color scheme based on chosen seed color. Uses the color api (https://www.thecolorapi.com/) to generate the color scheme shown.

## Tech Stack
- **Vite**
- **JavaScript**
- **Library**: React
- **Package Manager**: npm
